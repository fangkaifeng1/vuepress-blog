在计算机中，为什么0.1+0.2≠0.3？ 答：在计算机中存储数据都是以二进制来存的，像0.1、0.2这种属于浮点数，现在几乎所有语言都支持 IEEE 754 的二进制浮点格式。 常见的Math方法哪些？怎么用？ Math.random() 生成一个0-1之间的随机数 Math.floor() 函数用于向下取整，即将一个数字向下舍入到最接近的整数。 Math.round() 函数采用标准的四舍五入规则 Math.abs(x) : 返回 x 的绝对值。 Math.ceil(x) : 返回大于或等于 x 的最小整数。 Math.max(x1, x2, ..., xn) : 返回一组数中的最大值 Math.min(x1, x2, ..., xn) : 返回一组数中的最小值。 Math.pow(x, y) : 返回 x 的 y 次幂。
